# ENIGMA

A simple solution for sending one-time messages. The message can be read only once.

- aiohttp
- Redis

### Installation in system

1. install python3.6
2. install redis and run
3. clone this repo

```
cd app
pip3.7 install -r requirements.txt
```

### Start project
`python3.6 app.py`

### Run in podman\docker
podman-compose up -d
